REST API & Database Testing Framework

Project Overview

This repository contains 35+ test scenarios validating the backend REST APIs and database integrity for an E-Commerce application. The testing strategy ensures that frontend actions accurately reflect in the backend database, specifically targeting authentication, dashboard functionality, and data validation.

Tech Stack

API Testing Tool: Postman

Database: SQL

Defect Tracking: JIRA

Key Achievements

Validated REST APIs using Postman and verified backend database records through complex SQL queries.

Created 35+ robust test scenarios covering end-to-end data flow.

Identified and documented 12 critical defects (including silent database commit failures) with reproducible test evidence.

Collaborated closely with the development team to achieve a 100% bug closure rate.

Repository Structure

E_Commerce_API_Collection.json - Exported JSON collection of API requests and automated Javascript assertions.

Database_Validation_Queries.sql - SQL queries used to verify backend database integrity against API responses.
